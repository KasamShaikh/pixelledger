# Sample documents

Drop demo documents here. Suggested categories (one per type):

- `invoice.pdf` — printed invoice
- `handwritten_form.png` — scanned handwritten form
- `financial_table.pdf` — multi-column table
- `id.jpg` — ID / passport scan (use a sample, not real PII)
- `multipage.pdf` — 3+ pages
- `low_quality_scan.png` — skewed / noisy

Optionally add ground truth for accuracy metrics:

- `ground_truth/<filename>.txt` — plain text reference, OR
- `ground_truth/<filename>.json` — structured reference

The app accepts the ground-truth file via the sidebar uploader at run time.
